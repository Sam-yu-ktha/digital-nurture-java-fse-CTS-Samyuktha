# CTS Digital Nurture 5.0 - HTML5 Exercises

This repository contains the HTML5 exercise solutions completed as part of the Cognizant Digital Nurture 5.0 Java Full Stack Engineer (FSE) Upskilling Program.

## Exercises Completed

### Task 1 - HTML5 Base Template
- Created a basic HTML5 page structure
- Used semantic tags such as header, main, and footer

### Task 2 - Navigation and Linking
- Implemented navigation using anchor tags
- Added internal page links and external help page

### Task 3 - Welcome Message Styling
- Applied styling using id and class selectors
- Used internal CSS and inline CSS

### Task 4 - Community Event Gallery
- Displayed event images using tables
- Added image captions, alt text, and title attributes

### Task 5 - Event Registration Form
- Created a registration form using HTML5 form elements
- Implemented validation using required fields
- Used email, date, select, textarea, and output elements

### Task 6 - Event Handling
- Implemented JavaScript event handling
- Used onclick, onchange, onblur, ondblclick, and onkeyup events

### Task 7 - Video Invite
- Embedded a promotional video using the video element
- Implemented oncanplay and onbeforeunload events

### Task 8 - Saving User Preferences
- Used localStorage and sessionStorage
- Saved and retrieved user-selected event preferences

### Task 9 - Geolocation API
- Retrieved and displayed the user's current location
- Implemented geolocation error handling

### Task 10 - Debugging with Chrome DevTools
- Used console.log() for debugging
- Demonstrated breakpoints and variable inspection using DevTools

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Chrome DevTools
